
The project involves developing a Multi-Vector Phishing Detection System that serves as an intelligent security layer against social engineering by analyzing URLs, messages, and QR codes. The backend, powered by Python, utilizes a modular engine that performs Heuristic Analysis to detect structural anomalies in URLs, such as typosquatting or IP-based hosting, while integrating Reputation APIs (like VirusTotal) for real-time threat intelligence. For messages, the system employs Natural Language Processing (NLP) and ML to identify psychological triggers like urgency and uses computer vision libraries to decode visual links, ultimately aggregating these data points into a unified Risk Score delivered through a web-based dashboard.
